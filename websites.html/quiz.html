      Question 1. Which tag is used to define a paragraph in HTML?
              (A) <para>
              (B) <p>
              (C) <paragraph>
              (D) <pg>
          
Answers(B) <p>

    Question 2. To display an image on a webpage, which tag should be used?
            (A) <image> 
            (B) <img> 
            (C) <pic> 
            (D) <im>
          
Answers(B) <img>

    Question 3. What is the correct HTML tag for the largest heading?

            (B) <h6> 
            (C) <h1> 
            (D) <head>
          
Answers(C) <h1>

    Question 4. Which tag is used to create a table in HTML?

Answers(B) <HyperText>

    Question 11. What is a hyperlink in HTML?

    Question 12. Which of the following is true about running HTML files?
